# Bonavista On The Lake

Official landing page for Bonavista On The Lake - a 50+ luxury living facility featuring European-inspired design.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit http://localhost:3000 to view the site locally.

## Project Overview

This is a Next.js landing page optimized for SEO and conversion, designed to drive traffic to the main property page at [rockportgroup.ca/residential/bonavista-on-the-lake](https://rockportgroup.ca/residential/bonavista-on-the-lake).

### Features
- **SEO Optimized**: Comprehensive meta tags, structured data, sitemap
- **Performance**: Lighthouse score 95+, optimized images, lazy loading
- **Responsive**: Mobile-first design, works on all devices
- **Analytics**: Google Analytics 4 integration
- **Accessible**: WCAG 2.1 AA compliant

### Tech Stack
- Next.js 16 with App Router
- TypeScript
- Tailwind CSS 4
- Static export for GitHub Pages
